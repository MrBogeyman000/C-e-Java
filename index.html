Linguagens de Programação


Baixo Nível


Uma linguagem de baixo nível é o Assembly. Ela é tão próxima da linguagem de máquina que usa instruções básicas diretamente entendidas pelo processador. Em vez de lidar com variáveis e funções como em linguagens de alto nível, em Assembly, lidamos com registradores e endereços de memória.


Alto Nível


Java é um exemplo de linguagem de programação de alto nível, orientada a objeto. Ele capacita aplicativos, sistemas operacionais de smartphones, software empresarial e muitos programas conhecidos. Uma outra é o C++, de propósito geral, criada como uma extensão da linguagem C. Permite programação de alto nível e oferece controle direto sobre o hardware.
