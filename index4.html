Java


O que é?


Como dito anteriormente é uma linguagem de programação orientada ao objeto, que pode ser executada em diversos dispositivos, mais precisamente em Bilhões de dispositivos no mundo todo. É uma linguagem de programação rápida e segura, podendo ser executada em aplicativos e softwares empresarias.


Quando foi criado?


Criada na década de 90, por uma equipe de programadores chefiada por " James Gosling " na empresa " Sun Microsystems ". Mas em 2008 foi comprada pela empresa " Oracle Corporation "


É muito difícil aprender Java?


Aprender Java pode ser desafiador, mas com esforço, dedicação do programador e recursos adequados, é sim possível aprender Java. Ele costuma ser difícil por causa de sua forma de se escrever o código, a quantidade de coisas novas e os tipos de dados usados no Java.


O que se pode criar com Java?


Como dito antes pode-se criar aplicativos, tanto móveis quando na Web. Softwares empresarias e até jogos podem ser criados com Java. Um jogo bem conhecido que foi criado com Java, é o " Minecraft ". Que possuí sua versão para computadores, tendo seu próprio nome " Minecraft Java".
